Ben's 90s Nickeloden VHS Collection

I developed this project to hone my MySQL and Express.js skills. I built the database using MySQL Workbench and Amazon Web Services (AWS). I built the server using Express.js and Node.

I grew up during the heyday of 90s Nickelodeon, and I collect classic Nickelodeon VHS tapes. This website catalogues my collection. Enter a keyword in the search bar, like a show title, release date, or character to view relevant tapes in my collection. To view the complete list of VHS tapes, enter the search term "view all".
